    Get UnsuccessfulHealthCheck entries that are stale.

    Get UnsuccessfulHealthCheck entries that are stale. Stale recognition is based on the last update timestamp and counter. There need to be entries with the last update timestamp smaller or equal to the current Time - deviation. The counter needs to reach at least the minimum failed counter.

    deviation

    deviation expressed in milliseconds based on which the filtration of stale UnsuccessfulHealthChecks is carried out.

    minimumFailedCounters

    specified the minimum value of the counters we are looking for
